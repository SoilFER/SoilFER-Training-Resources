# Module 1: Introduction to R

 Data for exercises in Module 1 of the SoilFER training program:

  - *MIR_KANSAS_data.xlsx*: Soil Kansas dataset (KSSL) with site (lat-long and soil depths) and laboratory analyses (wet chemistry) and MIR spectral information.

 - *SOIL_UNITS_REFERENCE.txt*: Descriptive statistics for wet chemistry analises in the raw Kansas dataset.

 - *UNITS_QUICK_CARD.txt*: Quick reference card for soil units and conversions in the raw Kansas dataset.

  - *KSSL_site_lab* : Cleaned site and laboratory analyses (wet chemistry only) information for the KSSL dataset in Kansas, in two formats (.csv and .xlsx).
