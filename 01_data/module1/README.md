# Module 1: Introduction to R, Spatial Data and Data preparation for Digital Soil Mapping

 Data for exercises in Module 1 of the SoilFER training program:

  - *KSSL_data.xlsx*: Soil Kansas dataset (KSSL) with site (lat-long and soil depths) and laboratory analyses (wet chemistry).

  - *property_thresholds*: Table with feasible property thresholds for analytical parameters in the KSSL dataset.

 - *SOIL_UNITS_REFERENCE.txt*: Descriptive statistics for wet chemistry analyses in the raw Kansas dataset.

 - *UNITS_QUICK_CARD.txt*: Quick reference card for soil units and conversions in the raw Kansas dataset.
