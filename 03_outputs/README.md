# Data outputs from exercises in SoilFER training program Modules

Folder **module1**: Introduction to R, Spatial Data and Data preparation for Digital Soil Mapping

  - *KSSL_standardized*: Soil Kansas dataset (KSSL) with site (lat-long and soil depths) and percentiles of estimated laboratory analyses (wet chemistry) standardized to 0-30 adn 30-60 cm layers, in both `csv`and `.xlsx` file formats.
  
  - *KSSL_DSM_0-30*: Subset of the KSSL_standardized with median estimated laboratory analyses (wet chemistry) standardized to 0-30 cm layers, in both `csv`and `.xlsx` file formats.

 - *soil_property_validation_report.xlsx*: Report output from the validation of analytical parameters for wet chemistry analyses in the raw Kansas dataset.
